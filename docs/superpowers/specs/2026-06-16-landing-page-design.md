# Listen English Landing Page 设计文档

## 概述

为 Listen English 英语听力训练平台设计一个产品介绍落地页，实现于 `docs/introduce/index.html`，用于向潜在用户和已有用户展示产品核心功能。

## 约束条件

- **目标受众**：潜在用户 + 已有用户/展示用途
- **语言**：全中文
- **技术方案**：纯静态单文件 HTML，通过 CDN 引入 Tailwind CSS，零构建依赖
- **截图资源**：`./images/` 目录下的 7 张产品截图（1.png, 1.1.png, 1.2.png, 2.png, 3.png, 4.png, 5.png）
- **CTA 按钮**：页面顶部和底部各一个，链接到 `https://english.timetickme.com`

## 页面结构（6 个区块）

### 区块 1：Hero 区

- **标题**：Listen English
- **副标题**：基于真实语境的英语听力训练平台
- **说明文字**：精选 VOA 英语新闻素材，中英文字幕同步、智能查词、AI 语法分析，一站式提升英语听力水平
- **CTA 按钮**：「立即体验」→ `https://english.timetickme.com`
- **截图**：底部全宽展示 `images/1.png`，带轻微阴影（shadow-xl）和圆角（rounded-xl）
- **高度**：约 100vh
- **背景**：浅灰白 `#F8FAFC`

### 区块 2：核心特性概览

3 个等宽卡片横向排列：

| 卡片 | SVG 图标 | 标题             | 描述                                                         |
| ---- | -------- | ---------------- | ------------------------------------------------------------ |
| 1    | 耳机图标 | 沉浸式听力练习   | VOA 真实新闻素材，中英文字幕同步高亮，支持倍速播放和材料收藏 |
| 2    | 书本图标 | 智能查词与单词本 | 双击单词即查释义，自动收录到单词本，支持拼写练习巩固记忆     |
| 3    | AI 图标  | AI 句子分析      | 一键拆解长难句，分析句子结构、从句类型和语法成分             |

- **布局**：max-w-6xl 居中，3 列 grid
- **卡片样式**：白色背景，轻微阴影，圆角，图标使用 SVG 内联

### 区块 3：功能详情（3 组图文并排，左右交替）

#### 3a. 听力练习（左文右图）

- **截图**：`images/1.png`
- **标题**：真实语境，沉浸听力
- **要点**：
  - 200+ 篇 VOA 英语新闻素材，每日更新
  - 中英文字幕逐句同步高亮，跟读更轻松
  - 支持 0.5x ~ 2.0x 倍速调节
  - 收藏喜欢的材料，置顶快速访问

#### 3b. 智能查词 + 单词本（右文左图）

- **截图**：`images/1.1.png`
- **标题**：双击查词，即学即记
- **要点**：
  - 双击任意单词，右侧弹出 Collins 词典释义
  - 包含音标、发音、词组搭配和经典例句
  - 查过的单词自动进入单词本，随时复习
  - 拼写练习模式，根据释义拼写巩固记忆

#### 3c. AI 句子分析（左文右图）

- **截图**：`images/1.2.png`
- **标题**：AI 驱动，语法透视
- **要点**：
  - 点击句子旁 AI 按钮，智能分析句子结构
  - 拆解主语、谓语、宾语等核心成分
  - 标注从句类型和修饰关系
  - 逐层语法讲解，从整体到细节

**布局规则**：每组使用 flex 布局，文字区占 5/12，截图区占 7/12，第二组使用 `flex-row-reverse` 实现左右交替效果。截图带圆角和阴影。

### 区块 4：更多学习工具

横向 3 个小卡片，每个卡片上方截图缩略图、下方文字：

| 卡片 | 截图           | 标题     | 描述                                                  |
| ---- | -------------- | -------- | ----------------------------------------------------- |
| 1    | `images/3.png` | 单词搜索 | 手动输入单词查询 Collins 词典释义、词组搭配和经典例句 |
| 2    | `images/4.png` | 拼写练习 | 根据音标和释义拼写单词，20 题一组检验记忆效果         |

- **布局**：3 列等宽 grid
- **截图**：固定高度（约 200px），object-fit: cover，圆角

### 区块 5：技术栈

### 区块 6：底部 CTA

- **标题**：开始你的英语听力提升之旅
- **副标题**：免费使用，无需注册

## 响应式适配

- **桌面端（≥1024px）**：图文并排，3 列网格
- **平板端（768px-1023px）**：图文并排改为上下排列，3 列改为 2 列
- **移动端（<768px）**：单列垂直堆叠，Hero 截图缩小

## 技术实现要点

- 单文件 `docs/introduce/index.html`
- CDN 引入：`<script src="https://cdn.tailwindcss.com"></script>`
- SVG 图标内联（耳机、书本、AI 图标使用 Heroicons 风格简洁 SVG）
- 图片路径使用相对路径：`./images/1.png`
- HTML lang 属性设为 `zh-CN`
- 可选：`scroll-behavior: smooth` 实现平滑滚动
